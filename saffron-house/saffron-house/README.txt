Saffron House — Static Website
================================

Files:
  index.html   - main page
  styles.css   - all styling
  assets/      - images (hero, interior, dish1-3)

How to use:
  1. Unzip the folder.
  2. Double-click index.html to open in any browser.
  3. To publish: upload the entire folder contents to any static host
     (Netlify, Vercel, GitHub Pages, S3, your existing hosting, etc.)

Customization:
  - Edit text directly in index.html
  - Colors/fonts live in styles.css under :root { ... }
  - Replace images in assets/ keeping the same filenames
